📡 Sync with Mum

This lets Mum view your check-ins on her phone. Ask Mum to follow the steps below.

One-time setup (Mum does this): 1. Go to github.com → Settings → Developer settings → Personal access tokens → Tokens (classic)
2. Click Generate new token (classic)
3. Give it any name, e.g. checkin-app
4. Tick only gist under scopes
5. Click Generate token and copy it
6. Paste it into the box below and tap Save
